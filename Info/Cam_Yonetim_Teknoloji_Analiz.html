2.2 State Türleri

A. Ephemeral UI State (Sayfanın ömrü kadar)

Bu state'ler yalnızca bileşenin render edildiği sürece var olur:

const [kartlarAcik, setKartlarAcik] = useState(false)
const [sevkiyatAcik, setSevkiyatAcik] = useState(false)
const [notDuzenle, setNotDuzenle] = useState('')
const [dragSiparis, setDragSiparis] = useState<TakvimSiparis | null>(null)

Kullanım Alanları: Modal açılıp kapanması, UI interactionslar, temporary form states

B. Derived State (Başka state'lerden hesaplanan)

Başka state'lerden türetilmiş, yeniden hesaplanan data:

const selectedSiparisler = selectedDate 
  ? (siparisMap.get(selectedDate) ?? []) 
  : []
const monthGrid = buildMonthGrid()

Avantajı: Performance optimization, single source of truth

C. Data State (Supabase'den fetch edilen)

Server'dan gelen, kalıcı veri:

const [istatistikler, setIstatistikler] = useState<Istatistikler | null>(null)
const [takvimSiparisler, setTakvimSiparisler] = useState<TakvimSiparis[]>([])
const [yikamaBatchler, setYikamaBatchler] = useState<YikamaBatch[]>([])

D. Reference/Timer State (useRef ile)

Cleanup ve lifecycle management için:

const yikamaIntervalRef = useRef<ReturnType<typeof setInterval> | null>(null)
const dragNavTimerRef = useRef<ReturnType<typeof setTimeout> | null>(null)

3. Veri Akışı Patterns

3.1 Pattern: Initial Data Fetch + Polling

Dashboard'da kullanılıyor

1. Component Mount 2. useEffect tetiklenir 3. 7 parallel Supabase query 4. setState() ile veri güncellenir 5. Every 30 seconds: Yıkamada verisi polling 6. Cleanup: Interval'i clear et

3.2 Pattern: Calendar + Optimistic Update

Takvim sipariş sürükleme işleminde

1. Ay değişir → useEffect tetiklenir 2. Supabase: siparis + notları fetch et 3. setState({takvimSiparisler, takvimNotlari}) 4. Kullanıcı drag-drop işlemi başlatır 5. Optimistik: setTakvimSiparisler(new state) ← UI instant update 6. Background: Supabase .update().eq() ← sync

3.3 Pattern: Custom Hooks Encapsulation

// hooks/useStok.ts
export function useStok() {
  const [stoklar, setStoklar] = useState<Stok[]>([])
  const [yukleniyor, setYukleniyor] = useState(true)
  
  useEffect(() => {
    async function yukle() {
      const { data } = await supabase
        .from('stok')
        .select('*')
      setStoklar(data ?? [])
      setYukleniyor(false)
    }
    yukle()
  }, [])
  
  return { stoklar, yukleniyor }  // ← Encapsulated logic
}

// components/MyComponent.tsx
const { stoklar, yukleniyor } = useStok()

5.2 Validation Stack

// React Hook Form + Zod example
import { z } from 'zod'
import { zodResolver } from '@hookform/resolvers/zod'
import { useForm } from 'react-hook-form'

const siparisSchema = z.object({
  siparis_no: z.string().min(1, 'Sipariş no zorunlu'),
  teslim_tarihi: z.string()
    .refine(d => new Date(d) > new Date(), 'Geçmiş tarih seçilemez'),
  detaylar: z.array(z.object({
    adet: z.number().positive('Adet pozitif olmalı'),
  }))
})

type SiparisForm = z.infer<typeof siparisSchema>

const { register, formState: { errors } } = useForm<SiparisForm>({
  resolver: zodResolver(siparisSchema)
})

6. Supabase Integration

6.1 Direct Queries (No GraphQL/REST wrapper)

// lib/supabase.ts
import { createClient } from '@supabase/supabase-js'

export const supabase = createClient(
  import.meta.env.VITE_SUPABASE_URL,
  import.meta.env.VITE_SUPABASE_ANON_KEY
)

// Usage - Type-safe TypeScript client
const { data } = await supabase
  .from('siparisler')
  .select('id, siparis_no, cari(ad), siparis_detaylari(*)')
  .eq('durum', 'beklemede')
  .order('created_at', { ascending: false })

6.2 Özellikleri

  • ✓ Real-time subscriptions capable
  • ⚠ No request caching (manual management needed)
  • ⚠ No GraphQL complexity reduction

8. Önerilen İyileştirmeler

8.1 Global State Management

Eğer prop drilling sorun olursa şu seçenekler önerilir:

// Option 1: Zustand (lightweight)
import { create } from 'zustand'
const useAppStore = create(set => ({
  selectedDate: null,
  setSelectedDate: (date) => set({ selectedDate: date })
}))

// Option 2: Jotai (atomic)
import { atom, useAtom } from 'jotai'
const selectedDateAtom = atom<string | null>(null)
const [selectedDate, setSelectedDate] = useAtom(selectedDateAtom)

// Option 3: Context API + useReducer (built-in)
const AppContext = createContext<AppContextType>(initialState)
const [state, dispatch] = useReducer(reducer, initialState)

8.2 Request Caching

// React Query / TanStack Query (Recommended)
import { useQuery } from '@tanstack/react-query'

const { data: stoklar, isLoading } = useQuery({
  queryKey: ['stok'],
  queryFn: () => supabase.from('stok').select(),
  staleTime: 5 * 60 * 1000, // 5 minutes
  cacheTime: 10 * 60 * 1000, // 10 minutes
})
